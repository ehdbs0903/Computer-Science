# STL 컨테이너

STL은 Standard Template Library의 약자로 C++ 에서 자료구조와 알고리즘의 Generic Programming (일반화 프로그래밍) 개념을 구현하기 위해 개발 된 C++ 라이브러리
<br></br>

## 구성요소

- 반복자 : STL 컨테이너의 원소들을 가리키는 포인터
- 컨테이너: 데이터를 저장 및 관리를하는 클래스
- 알고리즘 : STL에서 제공하는 함수
<br></br>

## STL 컨테이너 종류

1. 시퀀스 컨테이너
	- 데이터를 선형적으로 저장
	- vector, list, dequeue …

2. 연관 컨테이너
	- 일정 규칙에 따라 조직화된 컨테이너
	- set, multiset, map …

3. 어댑터 컨테이너
	- 시퀀스나 연관 컨테이너를 변형하여 정해진 방식으로 관리
	- stack, queue …
